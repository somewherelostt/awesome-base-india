# Base India Circle - Product Overview

## Platform Mission
Connect Indian founders building on Base blockchain through intelligent discovery, real-time monitoring, and AI-powered insights.

## Core Components

### 1. Project Directory
- Smart project discovery with 3D carousel
- Live status indicators (Active, Building, Trending)
- Advanced filtering (category, batch, activity, tags)
- Global search with Cmd+K
- Real-time updates via WebSocket

### 2. Founder Connect Circle
- Interactive founder profiles with activity heatmaps
- Network visualization and collaboration tools
- Location-based discovery (India map)
- Skill matching for partnerships

### 3. AI Agent System
- **X Agent**: Monitors Twitter for launches, updates, engagement
- **GitHub Agent**: Tracks commits, deployments, code health
- **Orchestrator**: Coordinates workflows and notifications

## Key Metrics
- 12+ Projects on Base
- 12+ Indian Founders
- 5+ Batches/Programs
- Real-time activity tracking

## Technology Stack
- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS
- **Backend**: API Routes, PostgreSQL, Redis
- **AI**: OpenAI GPT-4, embeddings
- **Real-time**: WebSockets, BullMQ job queue
